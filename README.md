## 🍑Commit Rule
#### 📌study

- 공부한 내용 정리

- `[주제/study] yymmdd - 제목`  
  
  

#### 📌docs

- 개념 + 정의 정리
- 나중에 찾아보기 위해서 더 구체적으로 문서화하기
- `[주제/docs] 제목`  



#### 📌issue 

- 이슈 + 해결 모음
- `[주제/issue] 제목`
